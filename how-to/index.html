Bölüm 1

Admin Paneline Giriş

Admin paneli, tüm misafir işlemlerinin yönetildiği ana ekrandır. Sadece yetkili personel erişebilir.

Admin URL
thelifeco.app/admin
Giriş Yöntemi
E-posta + Şifre
Oturum Süresi
Tarayıcı kapatılana kadar (sessionStorage)

Giriş Adımları

  1. thelifeco.app/admin adresine gidin
  2. E-posta adresinizi girin (admin@thelifeco.app)
  3. Şifrenizi girin
  4. Sign In butonuna tıklayın
  5. Dashboard ekranı açılacaktır
Dikkat Tarayıcıyı kapattığınızda oturumunuz sona erer. Yeniden giriş yapmanız gerekir.
Bölüm 2

Dashboard (Ana Sayfa)

Dashboard, tesisin anlık durumunu gösteren kontrol panelidir.

İstatistik Kartları

  • Active Guests — Şu anda tesiste olan misafir sayısı
  • Upcoming — Yaklaşan check-in'ler
  • Occupancy — Doluluk oranı (oda sayısına göre)
  • Total Guests — Toplam kayıtlı misafir

Bugünkü Hareketler

  • Checking In Today — Bugün giriş yapacak misafirler
  • Checking Out Today — Bugün çıkış yapacak misafirler

Her misafirin adına tıklayarak detay sayfasına gidebilirsiniz.

Son Aktiviteler

Sağ tarafta son 6 işlem görüntülenir: misafir oluşturma, silme, ayar değişiklikleri vb.

Bölüm 3

Yeni Misafir Oluşturma

Her misafir için benzersiz bir portal oluşturulur. Bu portal, misafirin tüm program bilgilerini, terapi seçeneklerini ve tesis bilgilerini içerir.

Adım Adım Misafir Oluşturma

  1. Sol menüden Create Guest sekmesine tıklayın
  2. Misafir bilgilerini doldurun (aşağıdaki tabloya bakın)
  3. Sağ panelde programı oluşturun (günlük aktiviteler)
  4. Create Guest Portal butonuna tıklayın
  5. Ekranda görünen PIN kodunu mutlaka kaydedin
  6. Portal linkini ve PIN'i misafire WhatsApp ile gönderin

Form Alanları

Giriş tarihi

Program Oluşturma

Sağ paneldeki Build Manually sekmesinde günlük aktiviteleri ekleyin:

  1. Gün sekmesini seçin (Day 1, Day 2, ...)
  2. Saat girin (ör: 09:00)
  3. Aktivite adını yazın (ör: Morning Yoga, Lymphatic Drainage)
  4. Tür seçin: Group (grup), Food (yemek), Clinic (klinik), Extra (ekstra)
  5. + Add Activity ile yeni satır ekleyin
  6. Tüm günleri tamamlayın

Oluşturma Sonrası

Misafir başarıyla oluşturulduğunda ekranda şunlar görünür:

  • Portal URL — Misafirin erişeceği link (ör: thelifeco.app/guest/ad-soyad)
  • PIN — 4 haneli giriş kodu
  • Send WhatsApp Welcome — Hoş geldin mesajı gönderme butonu
  • Open Portal — Portalı yeni sekmede açma
Bölüm 4

Misafir Yönetimi

Sol menüdeki Guests sekmesinden tüm misafirleri görüntüleyebilir ve yönetebilirsiniz.

Misafir Listesi

  • Arama — İsim, e-posta, oda numarası veya telefon ile arayın
  • Filtreler — All / Active / Upcoming / Completed sekmeleri
  • Sıralama — İsim, check-in tarihi veya durum

Misafir Detay Sayfası

Bir misafirin adına tıklayarak detay sayfasını açabilirsiniz. Bu sayfada:

Hızlı İşlemler

  • Send WhatsApp Welcome — Hoş geldin mesajı gönder
  • Open Portal — Misafirin portalını aç
  • Edit Guest — Bilgileri düzenle
  • Mark Completed — Çıkış yapıldı olarak işaretle
  • Mark Active — Aktif olarak işaretle (check-in günü)
  • Reset PIN — Yeni PIN oluştur (eski PIN geçersiz olur)
  • Delete — Misafiri sil (geri alınamaz)

Bilgi Kartları

E-posta, WhatsApp, oda, check-in/out tarihleri, kalan gün sayısı, portal URL ve notlar.

Program Düzenleme

Detay sayfasındaki Edit Schedule butonuna tıklayarak mevcut programı düzenleyebilirsiniz. Değişiklikler Save Schedule ile kaydedilir.

Durum Değişiklikleri Misafir durumları otomatik olarak hesaplanır (check-in tarihine göre). Manuel olarak da değiştirebilirsiniz.

Aktivite Türleri

Grup aktiviteleri (yoga, pilates, meditasyon)

Program Düzenleme

  1. Misafir detay sayfasına gidin
  2. Edit Schedule butonuna tıklayın
  3. Gün sekmelerinden düzenlemek istediğiniz günü seçin
  4. Aktiviteleri ekleyin, silin veya değiştirin
  5. Save Schedule ile kaydedin
İpucu Sık kullanılan programlar için şablon oluşturun (Bölüm 6). Böylece her misafir için sıfırdan program yazmak zorunda kalmazsınız.
Bölüm 6

Program Şablonları

Şablonlar, tekrarlayan programları hızlıca uygulamanızı sağlar.

Şablon Oluşturma

  1. Sol menüden Programmes sekmesine gidin
  2. + New Template butonuna tıklayın
  3. Şablon adı, program adı ve süre girin
  4. Create Template ile kaydedin

Şablonu Misafire Uygulama

  1. Şablon satırındaki Apply to Guest butonuna tıklayın
  2. Açılan pencereden misafiri seçin
  3. Apply ile onaylayın
  4. Mevcut program silinir ve şablon uygulanır
Dikkat Şablon uygulamak, misafirin mevcut programını tamamen değiştirir. Eski program kaybolur.

Bölüm 12

WhatsApp Entegrasyonu

Admin Tarafı

  • Hoş geldin mesajı — Misafir oluşturulduğunda "Send WhatsApp Welcome" butonu
  • Mesaj içeriği: Portal linki + PIN kodu + karşılama metni
  • WhatsApp Web açılır, mesaj otomatik doldurulur, siz sadece gönder'e basarsınız

Misafir Tarafı

  • WhatsApp yardım butonu — Sağ üst köşede yeşil yuvarlak buton
  • Tıklayınca hızlı mesaj seçenekleri açılır:
    • Open Chat — Direkt sohbet aç
    • Running late — "Seansıma geç kalacağım"
    • Need assistance — "Yardıma ihtiyacım var"
    • Room service — "Oda servisi istiyorum"
  • Terapi rezervasyonu — Explore'daki "Request via WhatsApp" butonu ile doğrudan terapi planlayıcısına mesaj gönderilir
WhatsApp Numarası Ayarlar sayfasından WhatsApp numarasını değiştirebilirsiniz. Tüm portallar otomatik olarak güncellenir.

Bölüm 16

Güvenlik ve PIN Yönetimi

PIN Sistemi

  • Her misafir için benzersiz 4 haneli PIN oluşturulur
  • PIN, sunucuda şifrelenmiş (SHA-256 hash) olarak saklanır
  • PIN yalnızca oluşturma anında bir kez gösterilir
  • Kaybolursa Reset PIN ile yeni PIN oluşturulur

PIN Sıfırlama

  1. Misafir detay sayfasına gidin
  2. Reset PIN butonuna tıklayın
  3. Onay mesajını kabul edin
  4. Yeni PIN ekranda görünecektir — hemen kaydedin
  5. Yeni PIN'i misafire iletin

Güvenlik Önlemleri

  • Brute-force koruması — 5 yanlış PIN denemesinden sonra 15 dakika bekleme (sunucu taraflı)
  • Timing-safe karşılaştırma — PIN hash karşılaştırması zamanlama saldırılarına karşı korumalı
  • Oturum süresi — 24 saat sonra otomatik çıkış, 12 saat hard limit
  • HTTPS — Tüm iletişim şifrelidir
  • HSTS — Tarayıcılar her zaman HTTPS kullanmaya zorlanır
Asla Yapmayın PIN kodlarını e-posta ile göndermeyin. WhatsApp veya yüz yüze iletin. PIN'leri kağıda yazmayın.
Bölüm 17

PWA Kurulumu (Uygulama Gibi Kullanma)

Misafirler portalı telefonlarına uygulama olarak ekleyebilir. Bu sayede tam ekran deneyim yaşarlar.

iPhone (iOS) Kurulumu

  1. Safari'de portal linkini açın
  2. Alt çubuktaki Paylaş (kutu+ok) ikonuna dokunun
  3. "Ana Ekrana Ekle" seçeneğini seçin
  4. Ekle butonuna dokunun
  5. Ana ekranda TheLifeCo ikonu görünecektir

Android Kurulumu

  1. Chrome'da portal linkini açın
  2. Üstte çıkan "Uygulamayı Yükle" banner'ına dokunun
  3. Veya sağ üstteki üç nokta menüsünden "Ana ekrana ekle" seçin
  4. Yükle butonuna dokunun
Misafirlere Önerin Check-in sırasında misafirlere portalı ana ekrana eklemelerini önerin. Tam ekran deneyim çok daha kullanışlıdır ve çevrimdışı da çalışır.
Bölüm 18

Sık Sorulan Sorular

Misafir PIN'ini unutursa ne yapmalıyım?

Admin panelinden misafir detay sayfasına gidin ve Reset PIN butonuna tıklayın. Yeni PIN oluşturulur ve eski PIN geçersiz olur. Yeni PIN'i misafire hemen iletin.

Misafirin programını değiştirmem gerekiyor. Nasıl yapabilirim?

Misafir detay sayfasında Edit Schedule butonuna tıklayın. Mevcut program düzenleme modunda açılır. Değişikliklerinizi yapın ve Save Schedule ile kaydedin.

Yeni bir terapi eklemek istiyorum. Nasıl?

Şu anda Supabase panelinden therapies tablosuna yeni satır ekleyebilirsiniz. Gerekli alanlar: name, category_id, description. Opsiyonel: image_url, price_usd, duration, is_popular, is_recommended.

WhatsApp numarası değişti. Ne yapmalıyım?

Admin panelinde Settings sayfasına gidin. WhatsApp Number alanını güncelleyin ve Save Configuration ile kaydedin. Tüm portallarda otomatik güncellenir.

Misafir portalı çalışmıyor / yüklenmiyor.

Şu adımları kontrol edin:

  1. İnternet bağlantısını kontrol edin
  2. URL'nin doğru olduğundan emin olun: thelifeco.app/guest/ad-soyad
  3. Tarayıcı önbelleğini temizleyin
  4. Farklı bir tarayıcı deneyin
  5. Sorun devam ederse admin panelinden misafirin durumunu kontrol edin

Birden fazla admin kullanıcı ekleyebilir miyim?

Evet. Supabase Authentication panelinden yeni kullanıcı oluşturabilirsiniz. Yeni kullanıcının e-posta adresi, Vercel ortam değişkenlerindeki ADMIN_EMAILS listesine eklenmelidir.

Misafir check-out yaptıktan sonra portalına erişebilir mi?

Evet, portal erişilebilir durumda kalır. Misafir durumu otomatik olarak "Completed" olur ama veriler silinmez. İstenirse admin panelinden misafir silinebilir.

Veri yedeği alabilir miyim?

Admin panelinin Settings sayfasındaki Export JSON butonu ile mevcut verileri indirebilirsiniz. Ayrıca Supabase panelinden veritabanı yedeği alınabilir.
